QuakeMUD is a MUD dedicated to Quake. It is a place for deathmatchers to hang out, clans to meet, etc.

How to get a character

Just connect, login and follow the instructions.

Using QuakeMUD

  • connect using telnet.
  • view a listing of people who are online.
  • login to let the QuakeMUD Web Server know who you are. (this is important for some things)
Or, you can view QuakeMUD's extensive help documents.
QuakeMUD is running the BayWeb server by Phantom. Many thanks to Phantom for the use of his great server.
